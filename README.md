# HANK + Edgeworth Complementarity

Replication code and LaTeX source for:

> **On Heterogeneity and Edgeworth Complementarity Between Public and Private Consumption: Implications for the Fiscal Multiplier**
> Bright Quaye — Washington University in St. Louis
> Preliminary and incomplete. Do not circulate without permission.

## What's in the paper

We embed Edgeworth complementarity between public and private consumption into a tractable heterogeneous-agent New Keynesian (THANK) framework and derive closed-form fiscal multipliers with two new sufficient statistics: $(\chi, \xi)$. Under heterogeneous internalization ($\theta^H \neq \theta^S$), $\theta$-induced curvature heterogeneity decouples the aggregate MPC from the MPE, providing a new route to the Auclert–Bardóczy–Rognlie trilemma. A distributional dominance condition links inequality, complementarity, and the multiplier. Quantitative HANK calibration delivers impact multipliers in the range 1.68–1.84 versus 1.50 in the separable benchmark, with targeted fiscal policy raising the multiplier by another 24%.

## Quick start

### Build the paper

```bash
cd paper
pdflatex main
bibtex   main
pdflatex main
pdflatex main
```

### Reproduce all quantitative results and figures

```bash
# from repo root
python3 code/run_all.py
```

Individual scripts:

```bash
python3 code/simulation/analytical.py      # closed-form χ, ξ, μ
python3 code/simulation/figures.py         # all figures
python3 code/empirical/estimate_theta.py   # θ^H, θ^S from CEX
python3 code/hank/hank_jacobian.py         # HANK solver
```

Dependencies: `numpy`, `matplotlib`, `scipy`, `pandas`. Install via:

```bash
pip install numpy matplotlib scipy pandas
```

## Main quantitative results

| Case | χ | ξ | μ (THANK) | μ (HANK) |
|---|---|---|---|---|
| Bilbiie separable | 2.00 | 0.43 | 1.50 | 1.50 |
| Symmetric complementarity (θ = -0.5) | 2.00 | 0.30 | 1.36 | 1.41 |
| Symmetric substitutability (θ = +0.3) | 2.00 | 0.50 | 1.58 | 1.55 |
| **Heterogeneous internalization** (θ^S = 0.29, θ^H = -0.76) | 1.87 | 0.53 | 1.54 | **1.68** |
| Estimated upper bound (θ^S = 0.38, θ^H = -0.90) | 1.80 | 0.61 | 1.61 | **1.84** |
